CSS(Cascading Style Sheet)

CSS stands for Cascading Style Sheet. You can understand by the name that by using CSS, we can style our webpages using different properties and adding their values. We can change color of the text, font, font size, background color and much more.

Iniline, External and Internal CSS

In Inline CSS, you write the properties inside the opening tag of the element. Internal CSS is when you write the style properties in the head tag. Lastly, External CSS is when you write the style properties in a different file with the css extension.

Different types of Selectors

In CSS, there are 5 different types ofselectors. Namely, they are Element Selector, ID Selector, Class Selector, Universal Selector and Group Selector. Most of them are practiced in thiswebpage.
